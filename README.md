# Smart Health Clinic Queue & Eâ€‘Prescription Management System

A full-stack Java Spring Boot application for managing patient queues and e-prescriptions in a smart health clinic. Includes login/signup, role-based dashboards, appointment scheduling, prescription tracking, pharmacy integration, and Docker deployment.

## Features

- Patient registration and appointment booking
- Doctor queue management and prescription creation
- Pharmacy prescription tracking
- Admin dashboard for system overview
- Role-based access control
- Dockerized deployment
- GitHub version control
- Postman testing plan

## Technologies

- Java 17
- Spring Boot 3
- Thymeleaf
- MySQL
- Maven
- Docker
- GitHub

## Setup Instructions

1. Clone the repo  
2. Configure `application.properties`  
3. Run `mvn spring-boot:run`  
4. Access at `http://localhost:8080`
